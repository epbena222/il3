the basic structure of htlm is text inside of an opening tab and a closing tag such as h1, h2, h3, audio and image. these are all tags that can be used and there are several more that can be used.


below is listed the basic html tags these are used to make the basis of your webpage.

  • !DOCTYPE - Defines the doc type
  • html - defines an html section of the document
  • head - defines information about the webpage
  • title - defines the title
  • body - defines the body of the wepage
  • h1-h6 - HTML headings
  • p - defines a paragraph
  • br - inserts a one line break in the text
  • hr - defines a thematic content change
  • !--text-- - defines a comment
